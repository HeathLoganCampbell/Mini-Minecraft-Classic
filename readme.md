# Craft Classic
A Voxel based game that uses no libraries

## How to install
Just go to releases and download the jar

## How to play
```
W = forwards
S = backwards
A = left
D = right
left mouse button = break block
right mouse button = place block
right arrow key = changed block placed 
```
